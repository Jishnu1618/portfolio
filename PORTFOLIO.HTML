I enjoy building hands-on projects that combine innovation with real-world impact. Some of my key work includes: AgriGuru – a hackathon project using FastAPI, MongoDB, and CNN models to deliver intelligent crop and fertilizer recommendations, ranked 6th among 30 teams at an Infosys challenge. Virtual Barricading System – deployed YOLOv5 on Raspberry Pi 3 using custom datasets for real-time object detection and barricade monitoring. Image Editing WebApp – a full-stack project built with HTML, CSS, JavaScript, and FastAPI, integrated with the Cloudinary API for cloud-based storage and editing. Alongside projects, I am also improving in DSA, SQL, and cloud technologies (AWS, OCI) to strengthen my placement readiness. I bring to the table a mix of technical adaptability, problem-solving skills, and eagerness to learn, with the goal of pursuing opportunities in both software engineering and embedded electronics industries.
